Kein Download. Öffnen Sie ihn auf jedem Tablet- oder Handy-Browser und legen Sie los.

01Warum ein freihändiger Küchentimer?

Ein normaler Handy-Timer funktioniert gut für einen Topf, am Schreibtisch, mit trockenen Händen. Eine echte Küche ist nicht so.

💧

Ihre Hände sind nass, mehlig oder ölig — das Letzte, was Sie berühren wollen, ist ein Bildschirm.

🍳

Mehrere Dinge kochen gleichzeitig, und ein einziger eingebauter Timer reicht nicht aus.

⏳

Sie laufen ständig hin, nur um zu prüfen, "wie viel noch fehlt" bei etwas, das noch läuft.

🔁

Sie stellen dieselben Zeiten immer wieder ein — 3 Minuten für das hier, 8 für das dort — jeden einzelnen Tag.

02Sprachbefehle statt Knöpfe drücken

Jeder Befehl beginnt mit einem gesprochenen Aktivierungswort, damit er beim Küchenlärm nicht versehentlich ausgelöst wird.

Sagen Sie diesWas passiert
NEU
"Timer, Pasta 5 Minuten, Ramen 10 Minuten"		 Fügt zwei oder mehr Timer gleichzeitig hinzu und startet sie zusammen
NEU
"Timer, Ramen, wie viel Zeit bleibt?"		 Sagt die genaue verbleibende Zeit dieses Timers laut an
"Timer, Soba, 5 Minuten" Fügt einen neuen Timer hinzu und startet ihn sofort
"Timer, Stopp" Schaltet alle gerade klingelnden Alarme auf einmal stumm
"Timer, Udon, Pause" Pausiert einen bestimmten Timer anhand seines Namens
"Timer, Somen, zurücksetzen" Setzt einen bestimmten Timer auf seine ursprüngliche Zeit zurück

Sprachbefehle laufen mit deutscher Spracherkennung — sagen Sie das Aktivierungswort und die Befehle auf Deutsch, aber die Timer-Namen können alles sein: Pasta, Udon, Teigtaschen, Toast, was auch immer auf Ihrem Herd steht.

03Was ihn von einer normalen Timer-App unterscheidet

01

Mehrere Timer mit einem Satz hinzufügen

Name und Zeit, nebeneinander, in einem Atemzug — alle Timer starten gemeinsam.

"Timer, Teigtaschen 3 Min, Schwein 8 Min"
02

Fragen, wie viel Zeit noch bleibt

Kein Hinlaufen zum Bildschirm nötig — einfach fragen, und er antwortet laut.

"Ramen hat noch 7 Minuten 30 Sekunden übrig"
03

Favoriten mit nur einem Tipp

Markieren Sie eine Name-und-Zeit-Kombination einmal als Favorit, dann rufen Sie sie beim nächsten Mal mit nur einem Tipp wieder ab.

04

Alle Alarme mit einem Wort stoppen

Wenn mehrere Timer gleichzeitig klingeln, schaltet ein Befehl sie alle auf einmal stumm.

05

Individuelle Sprachansagen pro Timer

Statt eines allgemeinen Piepsens kann jeder Timer genau ansagen, wofür er ist.

"Die Teigtaschen sind fertig!"
06

Unbegrenzt viele Timer, Querformat-Layout

Fügen Sie so viele hinzu, wie Ihr Herd braucht, in einem Layout für ein an der Arbeitsplatte angelehntes Tablet.

04So funktioniert es

Tippen Sie darauf, oder sagen Sie es einfach — beides funktioniert gleich.

Timer starten

1. Auf "Timer Hinzufügen" tippen

Eine neue Karte erscheint, bereit zum Konfigurieren.

2. Namen und Voreinstellung antippen

Wählen Sie einen Namen aus dem Menü und eine Voreinstellung wie "5 Min", um die Dauer festzulegen.

🎙 oder sagen Sie einfach "Timer, Pasta 5 Minuten"

3. Auf "Start" tippen

Der Countdown beginnt. Per Sprache hinzugefügte Timer starten automatisch.

Timer beenden

1. Der Alarm meldet sich bei null

Er sagt den Timer beim Namen an — nicht nur ein Piepsen.

2. Auf der Karte "Stopp" tippen

Der Alarm verstummt und die Karte kehrt in den Bereitschaftsmodus zurück.

🎙 oder sagen Sie "Timer, Stopp", um alle klingelnden Timer stummzuschalten

3. Fertig

Drücken Sie "Zurücksetzen", um denselben Timer wieder von vorne laufen zu lassen.

Gemacht von einer echten Küche

Jikaseimen Kirinji (自家製麺キリンジ) ist ein Ramen-Restaurant in Sendai, Japan, spezialisiert auf Aburakasu-Ramen (frittierte Rinderinnereien) — das einzige Geschäft dieser Art in der Region Tohoku, seit rund 20 Jahren in Betrieb.

Der Freihändige Küchentimer wurde entwickelt, um ein echtes Problem dieser Küche zu lösen, und dann kostenlos für alle anderen freigegeben, die ihn brauchen.

05Häufig Gestellte Fragen

Ist der Freihändige Küchentimer kostenlos?
Ja. Er ist kostenlos, ohne Konto und ohne Installation — er läuft direkt im Browser.
Was macht ihn zu einem "freihändigen" Küchentimer?
Sie können Timer vollständig per Sprache starten, stoppen und prüfen, sodass Sie den Bildschirm nie mit nassen oder schmutzigen Händen berühren müssen.
Kann ich mehr als einen Timer gleichzeitig laufen lassen?
Ja. Fügen Sie so viele hinzu, wie Sie brauchen, und starten Sie mehrere auf einmal mit einem einzigen gesprochenen Satz.
Kann ich fragen, wie viel Zeit noch bleibt?
Ja. Fragen Sie einen Timer beim Namen ab, und er antwortet laut mit der verbleibenden Zeit.
Kann ich dieselben Einstellungen wiederverwenden, ohne sie neu einzugeben?
Ja. Markieren Sie einen Timer als Favorit, und starten Sie ihn dann mit demselben Namen und derselben Dauer wieder mit nur einem Tipp.
Piept der Alarm nur, oder kann er etwas Bestimmtes sagen?
Jeder Timer kann seine eigene Sprachansage haben, sodass er genau ansagt, was fertig ist, statt eines allgemeinen Piepsens.
Welche Sprache wird für die Sprachsteuerung unterstützt?
Sprachbefehle nutzen derzeit die deutsche Spracherkennung — das Aktivierungswort und die Befehle werden auf Deutsch gesprochen, während die Timer-Namen beliebig sein können.

Probieren Sie es in Ihrer Küche aus
